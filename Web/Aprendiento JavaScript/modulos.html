Modulos en JavaScript

Los módulos en JavaScript son una forma de dividir nuestro código en archivos separados, de tal forma que podamos importar y exportar funciones, variables, clases, etc. de un archivo a otro.

Para poder utilizar módulos en JavaScript, debemos de utilizar la directiva type="module" en la etiqueta script.

Modulo 1

js/modulos.js 

          
            export const PI = Math.PI;
            export function sumar(a, b) {
              return a + b;
            }
            export function restar(a, b) {
              return a - b;
            }

Modulo 2

js/no-module.js 

          
            import { PI, sumar, restar } from "./modulos.js";
            console.log(PI);
            console.log(sumar(5, 3));
            console.log(restar(5, 3));

Resultado

Resultado en consola 

          
            3.141592653589793
            8
            2

Usar alias

js/no-module.js 

          
            import { PI as PI2, sumar as sumar2, restar as restar2 } from "./modulos.js";
            console.log(PI2);
            console.log(sumar2(5, 3));
            console.log(restar2(5, 3));
