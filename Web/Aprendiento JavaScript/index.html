¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web interactivas. Es un lenguaje de programación del lado del cliente, lo que significa que se ejecuta en el navegador del usuario sin la necesidad de un servidor. JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos.

⬆️

Variables

Una variable es un contenedor para almacenar datos (valores). En JavaScript, se declaran variables utilizando la palabra clave var, let o const.

Tenemos dos tipos de datos en JavaScript:

  • Primitivos: string, number, boolean, null, undefined, symbol
  • Compuestos: object, array, function, class, etc.

var

La palabra clave var se utiliza para declarar una variable. Las variables declaradas con var pueden ser reasignadas y redeclaradas.

                    
                        var x = 10;
                        var y = 20;
                        var z = x + y;

let

La palabra clave let se utiliza para declarar una variable. Las variables declaradas con let pueden ser reasignadas, pero no redeclaradas.

                    
                        let x = 10;
                        let y = 20;
                        let z = x + y;

const

La palabra clave const se utiliza para declarar una constante. Las constantes no pueden ser reasignadas ni redeclaradas.

                    
                        const x = 10;
                        const y = 20;
                        const z = x + y;

Una constante no puede cambiar su valor después de la declaración. Si intenta reasignar una constante, se producirá un error a menos que la variable sea de un tipo de dato compuesto, en cuyo caso puede cambiar las propiedades del objeto o los elementos del array.

Nota: Se recomienda utilizar la palabra clave const siempre que sea posible, ya que las constantes son más seguras y fáciles de razonar que las variables. Si necesita reasignar una variable, utilice la palabra clave let.

Tipos de Variables

En JavaScript, las variables pueden almacenar diferentes tipos de datos, como números, cadenas, booleanos, objetos, arrays, funciones, clases, etc.

String

Una cadena es una secuencia de caracteres que se utilizan para representar texto. Las cadenas se pueden declarar utilizando comillas simples (''), comillas dobles ("") o comillas invertidas (``). 

                            
                                let nombre = 'Juan';
                                let apellido = "Pérez";
                                let nombreCompleto = `Juan Pérez`;

Los strings tienen dos propiedades y varios metodos que veremos a continuación.

Propiedades

Las propiedades son valores que se pueden obtener de las cadenas en JavaScript. Algunas de las propiedades más comunes son las siguientes:

  • length: Devuelve la longitud de la cadena.
                            
                                let nombre = 'Juan';
                                console.log(nombre.length); // 4
  • prototype: Devuelve el prototipo de la cadena.
                            
                                let nombre = 'Juan';
                                console.log(nombre.prototype); // String

Metodos

Los metodos son funciones que se pueden utilizar con las cadenas en JavaScript. Algunos de los métodos más comunes son los siguientes:

  • toUpperCase:Devuelve la cadena en mayúsculas.
                            
                                let nombre = 'Juan';
                                console.log(nombre.toUpperCase()); // JUAN
  • toLowerCase:Devuelve la cadena en minúsculas.
                            
                                let nombre = 'Juan';
                                console.log(nombre.toLowerCase()); // juan
  • charAt:Devuelve el carácter en la posición especificada.
                            
                                let nombre = 'Juan';
                                console.log(nombre.charAt(0)); // J
  • includes:Devuelve true si la cadena contiene el texto especificado, de lo contrario devuelve false.
                            
                                let nombre = 'Juan';
                                console.log(nombre.includes('J')); // true
  • indexOf:Devuelve la posición de la primera aparición del texto especificado en la cadena.
                            
                                let nombre = 'Juan';
                                console.log(nombre.indexOf('a')); // 1
  • lastIndexOf:Devuelve la posición de la última aparición del texto especificado en la cadena.
                            
                                let nombre = 'Juan';
                                console.log(nombre.lastIndexOf('a')); // 3
  • replace:Reemplaza el texto especificado en la cadena.
                            
                                let nombre = 'Juan';
                                console.log(nombre.replace('Juan', 'Pedro')); // Pedro
  • slice:Extrae una parte de la cadena y devuelve una nueva cadena.
                            
                                let nombre = 'Juan';
                                console.log(nombre.slice(0, 2)); // Ju
  • split:Divide la cadena en un array de subcadenas.
                            
                                let nombreCompleto = 'Juan Pérez';
                                console.log(nombreCompleto.split(' ')); // ['Juan', 'Pérez']
  • substr:Extrae una parte de la cadena, comenzando en la posición especificada.
                            
                                let nombre = 'Juan';
                                console.log(nombre.substr(0, 2)); // Ju
  • trim:Elimina los espacios en blanco al principio y al final de la cadena.
                            
                                let nombre = ' Juan ';
                                console.log(nombre.trim()); // Juan

Estos son solo algunos de los métodos que se pueden utilizar con strings en JavaScript. Para obtener más información sobre los métodos de las cadenas, consulte la documentación oficial de JavaScript.

Template strings

Los template strings son una forma de crear cadenas de texto en JavaScript que permiten incrustar expresiones dentro de ellas. Los template strings se crean utilizando comillas invertidas (``) en lugar de comillas simples ('') o comillas dobles ("").

Para entender lo que vamos a ver necesitamos saber que son las concatenaciones y las interpolaciones.

Concatenación: La concatenación es la unión de dos o más cadenas en una sola cadena. En JavaScript, se utiliza el operador + para concatenar cadenas. 

                            
                                let nombre = 'Juan';
                                let apellido = 'Pérez';
                                let nombreCompleto = nombre + ' ' + apellido;
                                console.log(nombreCompleto); // Juan Pérez

Esto puede llegar a ser bastante tedioso si tenemos que concatenar muchas cadenas. Para evitar esto, podemos utilizar los template strings y aquí es donde entra en juego la interpolación.

Interpolación: La interpolación es una forma de insertar expresiones dentro de una cadena. En JavaScript, se utiliza el signo de dólar y llaves (${expresión}) para interpolar expresiones dentro de los template strings. 

                            
                                let nombre = 'Juan';
                                let apellido = 'Pérez';
                                let nombreCompleto = `Hola mi nombre es ${nombre} ${apellido}.`;
                                console.log(nombreCompleto); // Juan Pérez

Otra cosa que podemos hacer es respetar el identado y los saltos de línea con el template. 

                            
                                let nombre = 'Juan';
                                let apellido = 'Pérez';
                                let nombreCompleto = `
                                    Hola mi nombre es ${nombre} ${apellido}.
                                    ¿Cómo estás?
                                `;
                                console.log(nombreCompleto);
                                // Hola mi nombre es Juan Pérez.
                                // ¿Cómo estás?

Number

Un número es un valor numérico que se utiliza para representar cantidades. Los números se pueden declarar utilizando números enteros, números decimales, números exponenciales, etc.

Podemos declararlos de varias maneras: 

                        
                            let a = 2 //declaración de un número entero (mejor manera)
                            let b = new Number(3) //declaración entero con su constructor
                            let c = 3.14 //declaración de un número decimal

Metodos

Los números tienen varios Metodos que se pueden utilizar para realizar operaciones matemáticas en JavaScript. Algunos de los prototipos más comunes son los siguientes:

  • toExponential:Devuelve un dato que representa el número en notación exponencial.
                            
                                let a = 2;
                                console.log(a.toExponential()); // 2e+0
  • toFixed:Devuelve un dato que representa el número con un número fijo de decimales.
                            
                                let c = 3.141592653589793;
                                console.log(c.toFixed(2)); // 3.14
  • toPrecision:Devuelve un dato que representa el número con una precisión específica.
                            
                                let c = 3.141592653589793;
                                console.log(c.toPrecision(2)); // 3.1
  • toString:Devuelve un dato que representa el número como una cadena.
                            
                                let c = 3.141592653589793;
                                console.log(c.toString()); // 3.141592653589793
  • valueOf:Devuelve el valor primitivo del número.
                            
                                let a = 2;
                                console.log(a.valueOf()); // 2
  • parseInt: Devuelve solo la parte entera de un número flotante o si el dato es de tipo cadena devuelve el número entero. 
                            
                                let d = '2.5', let c = 3,141592653589793
                                console.log(parseInt(d)); // 2
                                console.log(parseInt(c)); // 3
  • parseFloat: Devuelve el número flotante de un número entero o si el dato es de tipo cadena devuelve el número flotante. 
                            
                                let d = '2.5', let c = 3,141592653589793
                                console.log(parseFloat(c)); // 3.141592653589793
                                console.log(parseFloat(d)); // 2.5

Boolean

Un booleano es un valor lógico que se utiliza para representar verdadero o falso. Los booleanos se pueden declarar utilizando los valores true o false.

Podemos declararlos de varias maneras: 

                        
                            let verdadero = true //declaración de un booleano verdadero
                            let falso = false //declaración de un booleano falso
                            let v = Boolean(1) //declaración de un booleano verdadero con su constructor
                            let f = Boolean(0) //declaración de un booleano falso con su constructor

Existen algunos valores que se consideran verdaderos y otros que se consideran falsos en JavaScript.

  • True:
    • (true)
    • ({})
    • ([])
    • (42)
    • ("foo")
    • (new Date())
    • (-42)
    • (3.14)
    • (-3,14)
    • (Infinity)
    • (-Infinity)
  • False:
    • (false)
    • (null)
    • (undefined)
    • (0)
    • (-0)
    • (0n)
    • ("")
    • ('')
    • (``)
    • (NaN)

Undefined, Null y NaN

Undefined

Undefined es un valor primitivo que se utiliza para representar algo que no está definido o que no tiene un valor asignado. En JavaScript, una variable se inicializa con undefined si no se le asigna un valor. 

                        
                            let x;
                            console.log(x); // undefined

Null

Null es un valor primitivo que se utiliza para representar la ausencia de un valor. En JavaScript, null se utiliza para indicar que una variable no tiene un valor asignado. El programador puede asignar null de manera intencional para indicar que una variable no tiene un valor. 

                        
                            let y = null;
                            console.log(x); // null

NaN

NaN es un valor primitivo que se utiliza para representar un valor que no es un número. En JavaScript, NaN se utiliza para indicar que una operación matemática no se puede realizar. 

                        
                            let z = 10 / 'a';
                            console.log(z); // NaN

Funciones

Una función es un bloque de código que se puede reutilizar para realizar una tarea específica. En JavaScript, las funciones se pueden declarar utilizando la palabra clave function.

Podemos declarar funciones de dos maneras:

Función Declarada

Una función declarada es una función que se declara utilizando la palabra clave function seguida de un nombre y un cuerpo de función. 

                        
                            function saludar() {
                                console.log('Hola Mundo!');
                            }
                            saludar(); // Hola Mundo!

Las funciones declaradas se pueden llamar antes de que se declaren en el código, ya que JavaScript carga primero las funciones en la memoria y luego ejecuta el código.

Funciones Expresada

Una función expresada es una función que se asigna a una variable o constante. Las funciones expresadas no se pueden llamar antes de que se declaren en el código, ya que JavaScript carga primero las variables y luego ejecuta el código. 

                        
                            let saludar1 = function() {
                                console.log('Hola Mundo!');
                            }

Las funciones expresadas se pueden llamar después de que se declaren en el código, ya que JavaScript carga primero las variables y luego ejecuta el código.

Retorno

Una función puede devolver un valor utilizando la palabra clave return. El valor devuelto por una función se puede asignar a una variable o constante. 

                        
                            function sumar(a, b) {
                                return a + b;
                            }
                            let resultado = sumar(2, 3);
                            console.log(resultado); // 5

Parámetros

Una función puede tener parámetros que se utilizan para pasar valores a la función. Los parámetros se definen entre paréntesis y se separan por comas. 

                        
                            function saludar(nombre) {
                                console.log(`Hola ${nombre}!`);
                            }
                            saludar('Juan'); // Hola Juan!

Arreglos

Un arreglo es una colección de elementos que se utilizan para almacenar múltiples valores en una sola variable. En JavaScript, los arreglos se pueden declarar de varias formas y los elementos se separan por comas.

Estas son algunas maneras de declarar arreglos en JavaScript:

Arreglo Literal

Un arreglo literal es una forma de declarar un arreglo utilizando corchetes [] y separando los elementos por comas. 

                        
                            let colores = ['rojo', 'verde', 'azul'];
                            console.log(colores); // ['rojo', 'verde', 'azul']

Arreglo Constructor

Un arreglo constructor es una forma de declarar un arreglo utilizando la palabra clave new seguida de la clase Array y los elementos entre paréntesis. 

                        
                            let colores = new Array('rojo', 'verde', 'azul');
                            console.log(colores); // ['rojo', 'verde', 'azul']

Propiedades y Métodos

Los arreglos en JavaScript tienen varias propiedades y métodos que se pueden utilizar para trabajar con ellos. Algunas de las propiedades y métodos más comunes son los siguientes:

Propiedades

Las propiedades son valores que se pueden obtener de los arreglos en JavaScript. Algunas de las propiedades más comunes son las siguientes:

  • length: Devuelve la longitud del arreglo.
                            
                                let colores = ['rojo', 'verde', 'azul'];
                                console.log(colores.length); // 3
  • prototype: Devuelve el prototipo del arreglo.
                            
                                let colores = ['rojo', 'verde', 'azul'];
                                console.log(colores.prototype); // Array

Métodos

Los métodos son funciones que se pueden utilizar con los arreglos en JavaScript. Algunos de los métodos más comunes son los siguientes:

  • push: Agrega un elemento al final del arreglo.
                            
                                let colores = ['rojo', 'verde', 'azul'];
                                colores.push('amarillo');
                                console.log(colores); // ['rojo', 'verde', 'azul', 'amarillo']
  • pop: Elimina el último elemento del arreglo.
                            
                                let colores = {'rojo', 'verde', 'azul'}
                                colores.pop();
  • shift: Elimina el primer elemento del arreglo.
                            
                                let colores = ['rojo', 'verde', 'azul'];
                                colores.shift();
                                console.log(colores); // ['verde', 'azul']
  • unshift: Agrega un elemento al principio del arreglo.
                            
                                let colores = ['rojo', 'verde', 'azul'];
                                colores.unshift('amarillo');
                                console.log(colores); // ['amarillo', 'rojo', 'verde', 'azul']
  • join: Convierte los elementos del arreglo en una cadena separada por comas.
                            
                                let colores = ['rojo', 'verde', 'azul'];
                                console.log(colores.join()); // rojo,verde,azul
  • reverse: Invierte el orden de los elementos del arreglo.
                            
                                let colores = ['rojo', 'verde', 'azul'];
                                colores.reverse();
                                console.log(colores); // ['azul', 'verde', 'rojo']
  • sort: Ordena los elementos del arreglo en orden alfabético o numérico.
                            
                                let colores = ['rojo', 'verde', 'azul'];
                                colores.sort();
                                console.log(colores); // ['azul', 'rojo', 'verde']
                                let numeros = [3, 1, 2];
                                numeros.sort();
                                console.log(numeros); // [1, 2, 3]
  • forEach: Recorre un arreglo.
                            
                                let colores = ['rojo', 'verde', 'azul'];
                                colores.forEach(function(elemento) {
                                    console.log(elemento);
                                });
                                // rojo
                                // verde
                                // azul

Estos son solo algunos de los métodos que se pueden utilizar con arreglos en JavaScript. Para obtener más información sobre los métodos de los arreglos, consulte la documentación oficial de JavaScript.

Objetos

Un objeto es una colección de propiedades y métodos que se utilizan para representar un objeto del mundo real en JavaScript. En JavaScript, los objetos se pueden declarar de varias formas y las propiedades y métodos se separan por comas.

Objeto Literal

Un objeto literal es una forma de declarar un objeto utilizando llaves {} y separando las propiedades y métodos por comas. 

                        
                            let persona = {
                                nombre: 'Juan', 
                                apellido: 'Peréz',
                                edad: 30,
                                colores_favoritos: ['rojo', 'verde', 'azul'],
                                saludar: function(){
                                    console.log(`Hola mi nombre es ${this.nombre} ${this.apellido}, tengo ${this.edad} años.)

                                }
                            }
                            console.log(persona); // {nombre: 'Juan', apellido: 'Peréz', edad: 30}

Objeto Constructor

Un objeto constructor es una forma de declarar un objeto utilizando la palabra clave new seguida de la clase Object y las propiedades y métodos entre paréntesis. 

                        
                            let persona = new Object();
                            persona.nombre = 'Juan';
                            persona.apellido = 'Peréz';
                            persona.edad = 30;
                            persona.colores_favoritos = ['rojo', 'verde', 'azul'];
                            persona.saludar = function(){
                                console.log(`Hola mi nombre es ${this.nombre} ${this.apellido}, tengo ${this.edad} años.`)
                            }
                            console.log(persona); // {nombre: 'Juan', apellido: 'Peréz', edad: 30}

Propiedades y Métodos

Los objetos en JavaScript tienen varias propiedades y métodos que se pueden utilizar para trabajar con ellos. Algunas de las propiedades y métodos más comunes son los siguientes:

Propiedades

Las propiedades son valores que se pueden obtener de los objetos en JavaScript. Algunas de las propiedades más comunes son las siguientes:

  • constructor: Devuelve la función constructora del objeto.
                            
                                let persona = {
                                    nombre: 'Juan', 
                                    apellido: 'Peréz',
                                    edad: 30,
                                    colores_favoritos: ['rojo', 'verde', 'azul'],
                                    saludar: function(){
                                        console.log(`Hola mi nombre es ${this.nombre} ${this.apellido}, tengo ${this.edad} años.`)
                                    }
                                }
                                console.log(persona.constructor); // Object
  • prototype: Devuelve el prototipo del objeto.
                            
                                let persona = {
                                    nombre: 'Juan', 
                                    apellido: 'Peréz',
                                    edad: 30,
                                    colores_favoritos: ['rojo', 'verde', 'azul'],
                                    saludar: function(){
                                        console.log(`Hola mi nombre es ${this.nombre} ${this.apellido}, tengo ${this.edad} años.`)
                                    }
                                }
                                console.log(persona.prototype); // Object

Métodos

Los métodos son funciones que se pueden utilizar con los objetos en JavaScript. Algunos de los métodos más comunes son los siguientes:

  • hasOwnProperty: Devuelve true si el objeto tiene la propiedad especificada, de lo contrario devuelve false. 
                            
                                let persona = {
                                    nombre: 'Juan', 
                                    apellido: 'Peréz',
                                    edad: 30,
                                    colores_favoritos: ['rojo', 'verde', 'azul'],
                                    saludar: function(){
                                        console.log(`Hola mi nombre es ${this.nombre} ${this.apellido}, tengo ${this.edad} años.`)
                                    }
                                }
                                console.log(persona.hasOwnProperty('nombre')); // true
  • isPrototypeOf: Devuelve true si el objeto es un prototipo del objeto especificado, de lo contrario devuelve false. 
                                
                                    let persona = {
                                        nombre: 'Juan', 
                                        apellido: 'Peréz',
                                        edad: 30,
                                        colores_favoritos: ['rojo', 'verde', 'azul'],
                                        saludar: function(){
                                            console.log(`Hola mi nombre es ${this.nombre} ${this.apellido}, tengo ${this.edad} años.`)
                                        }
                                    }
                                    console.log(persona.isPrototypeOf('nombre')); // false
  • propertyIsEnumerable: Devuelve true si la propiedad especificada es enumerable, de lo contrario devuelve false. 
                            
                                let persona = {
                                 nombre: 'Juan', 
                                 apellido: 'Peréz',
                                 edad: 30,
                                 colores_favoritos: ['rojo', 'verde', 'azul'],
                                    saludar: function(){
                                         console.log(`Hola mi nombre es ${this.nombre} ${this.apellido}, tengo ${this.edad} años.`)
                                        }
                                   }
                                 console.log(persona.propertyIsEnumerable('nombre')); // true

Estos son solo algunos de los métodos que se pueden utilizar con objetos en JavaScript. Para obtener más información sobre los métodos de los objetos, consulte la documentación oficial de JavaScript.

Si queremos acceder a las propiedades de ciertor objetos o cambiar ciertos valores podemos hacerlo de la siguiente manera: 

                     
                        let persona = {
                            nombre: 'Juan', 
                            apellido: 'Peréz',
                            edad: 30,
                            colores_favoritos: ['rojo', 'verde', 'azul'],
                            saludar: function(){
                                console.log(`Hola mi nombre es ${this.nombre} ${this.apellido}, tengo ${this.edad} años.`)
                            }
                        }
                        console.log(persona.nombre); // Juan
                        console.log(persona['nombre']); // Juan
                        persona.nombre = 'Pedro'; // Cambiar el valor de la propiedad nombre
                        console.log(persona.nombre); // Pedro 
                        persona['nombre'] = 'Juan';
                        console.log(persona.nombre); // Juan

operadores

Los operadores son símbolos que se utilizan para realizar operaciones en JavaScript. Los operadores se pueden clasificar en varios tipos, como operadores aritméticos, operadores de asignación, operadores de comparación, operadores lógicos, operadores de tipo, operadores de bits, etc. Veremos algunos de estos tipos de operadores a continuación.

Operadores Aritmeticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas en JavaScript. Algunos de los operadores aritméticos más comunes son los siguientes:

  • + Suma dos valores.
                        
                            let a = 2, b = 3;
                            console.log(a + b); // 5
  • - Resta dos valores.
                        
                            let a = 2, b = 3;
                            console.log(a - b); // -1
  • * Multiplica dos valores.
                          
                            let a = 2, b = 3;
                            console.log(a * b); // 6
  • / Divide dos valores.
                        
                            let a = 6, b = 3;
                            console.log(a / b); // 2
  • % Devuelve el resto de la división de dos valores.
                        
                            let a = 6, b = 3;
                            console.log(a % b); // 0
  • ++ Incrementa en uno el valor de una variable.
                        
                            let a = 2;
                            a++;
                            console.log(a); // 3
  • -- Decrementa en uno el valor de una variable.
                        
                            let b = 3;
                            b--;
                            console.log(b); // 2
  • ** Eleva un número a una potencia.
                        
                            let a = 2, b = 3;
                            console.log(a ** b); // 8

Relacionales

Los operadores relacionales se utilizan para comparar dos valores en JavaScript. Algunos de los operadores relacionales más comunes son los siguientes:

  • == Comprueba si dos valores son iguales.
                        
                            let a = 2, b = 3;
                            console.log(a == b); // false
  • != Comprueba si dos valores son diferentes.
                        
                            let a = 2, b = 3;
                            console.log(a != b); // true
  • > Comprueba si el primer valor es mayor que el segundo valor. 
                        
                            let a = 2, b = 3;
                            console.log(a > b); // false
  • < Comprueba si el primer valor es menor que el segundo valor. 
                        
                            let a = 2, b = 3;
                            console.log(a < b); // true
  • >= Comprueba si el primer valor es mayor o igual que el segundo valor. 
                        
                            let a = 2, b = 3;
                            console.log(a >= b); // false
  • <= Comprueba si el primer valor es menor o igual que el segundo valor. 
                        
                            let a = 2, b = 3;
                            console.log(a <= b); // true
  • === Comprueba si dos valores son iguales y del mismo tipo. 
                        
                            let a = 2, b = '2';
                            console.log(a === b); // false
  • !== Comprueba si dos valores son diferentes o no del mismo tipo. 
                         
                            let a = 2, b = '2';
                            console.log(a !== b); // true

Operadores Lógicos

Los operadores lógicos se utilizan para realizar operaciones lógicas en JavaScript. Algunos de los operadores lógicos más comunes son los siguientes:

  • &&(and) Devuelve true si ambos valores son verdaderos. 
                            
                                let a = true, b = false;
                                console.log(a && b); // false
  • ||(or) Devuelve true si al menos uno de los valores es verdadero. 
                            
                                let a = true, b = false;
                                console.log(a || b); // true
  • !(not) Devuelve true si el valor es falso y false si el valor es verdadero. 
                            
                                let a = true;
                                console.log(!a); // false

Estructuras de Control

Las estructuras de control se utilizan para controlar el flujo de un programa en JavaScript. Las estructuras de control se pueden clasificar en varias categorías, como estructuras de control condicionales, estructuras de control de bucles, estructuras de control de salto, etc. Veremos algunos ejemplos de estas estructuras de control a continuación.

Condicionales

Las estructuras de control condicionales se utilizan para tomar decisiones en JavaScript. Algunas de las estructuras de control condicionales más comunes son los siguientes:

If

La estructura de control if se utiliza para ejecutar un bloque de código si una condición es verdadera. 

                            
                                let edad = 18;
                                if (edad >= 18) {
                                    console.log('Eres mayor de edad.');
                                }

Else

La estructura de control else se utiliza para ejecutar un bloque de código si una condición es falsa. 

                            
                                let edad = 16;
                                if (edad >= 18) {
                                    console.log('Eres mayor de edad.');
                                } else {
                                    console.log('Eres menor de edad.');
                                }

If Else If Else

La estructura de control if else if else se utilza generalmente cuando tenemos más de dos condiciones. por ejemplo si quremos que la consola imprima un mensaje dependiendo de la hora del día. 

                            
                                let hora = 14;
                                if (hora < 12) {
                                    console.log('Buenos días.');
                                } else if (hora < 18) {
                                    console.log('Buenas tardes.');
                                } else {
                                    console.log('Buenas noches.');
                                }

Operador Ternario

El operador ternario se utiliza para simplificar la estructura de control if else en una sola línea. 

                            
                                let edad = 16;
                                let mensaje = (edad >= 18) ? 'Eres mayor de edad.' : 'Eres menor de edad.';
                                console.log(mensaje);

Switch

La estructura de control switch se utiliza para ejecutar un bloque de código dependiendo del valor de una variable.

Ejemplo: Si queremos que la consola imprima un mensaje dependiendo del día de la semana. 

                            
                                let dia = 1;
                                switch (dia) {
                                    case 1:
                                        console.log('Lunes');
                                        break;
                                    case 2:
                                        console.log('Martes');
                                        break;
                                    case 3:
                                        console.log('Miércoles');
                                        break;
                                    case 4:
                                        console.log('Jueves');
                                        break;
                                    case 5:
                                        console.log('Viernes');
                                        break;
                                    case 6:
                                        console.log('Sábado');
                                        break;
                                    case 7:
                                        console.log('Domingo');
                                        break;
                                    default:
                                        console.log('Día inválido');
                                }

Bucles (Loops)

Los bucles (loops) se utilizan para repetir un bloque de código en JavaScript. Los bucles se pueden clasificar en varias categorías, como bucles for, bucles while, bucles do while, etc. A continuación veremos algunos de los bucles más comunes en JavaScript.

while

El bucle while se utiliza para repetir un bloque de código mientras una condición sea verdadera. Por ejemplo si queremos que la consola imprima los números del 1 al 10. 

                        
                            let i = 1;
                            while(i < 11){
                                console.log(i)
                                i++
                            }

do while

El bucle do while se utiliza para iterar al menos una vez las instrucciones dentro del bucle y luego evaluar la condicion. Por ejemplo si queremos que la consola imprima los números del 1 al 10 

                        
                            let i = 1;
                            do {
                                console.log(i)
                                i++
                            } while(i < 11);

for

El bucle for se utiliza para repetir un bloque de código un número específico de veces. Por ejemplo si queremos que la consola imprima los números del 1 al 10. 

                        
                            for (let i = 1; i < 11; i++)
                            {
                                console.log(i);
                            }

forIn

El bucle for in se utiliza para iterar sobre las propiedades de un objeto. Por ejemplo si queremos que la consola imprima las propiedades y también sus valores de un objeto. 

                        
                            let persona = {
                                nombre: 'Juan',
                                apellido: 'Pérez',
                                edad: 30
                            };
                            for (let propiedadValor in persona) {
                                console.log(`${propiedadValor}:${persona[propiedadValor]}`);
                            };

forOf

El bucle for of se utiliza para iterar sobre los elementos de un arreglo. Por ejemplo si queremos que la consola imprima los elementos de un arreglo. 

                        
                            let numeros = [1, 2, 3, 4, 5];
                            forOf (let numero of numeros) {
                                console.log(numero);
                            }

El bloque catch se utiliza para manejar los errores que se producen en el bloque try. El bloque finally se utiliza para ejecutar un bloque de código después de que se haya ejecutado el bloque try y el bloque catch. Por ejemplo si el dato que esperamos recibir es un número y el usuario ingresa un texto podemos tirar nuestro error personalizado.

Destructuracion

La destructuración es una característica de JavaScript que permite extraer valores de arreglos y objetos y asignarlos a variables. La destructuración se puede utilizar con arreglos y objetos en JavaScript. 

                
                    const marcas = ['Toyota', 'Honda', 'Nissan'];
                    let [marca1, marca2, marca3] = marcas;
                    console.log(marca1); // Toyota
                    console.log(marca2); // Honda
                    console.log(marca3); // Nissan
                
            
                
                    let persona = {
                        nombre: 'Juan',
                        apellido: 'Pérez',
                        edad: 30
                    };
                    let {nombre, apellido, edad} = persona;
                    console.log(nombre); // Juan
                    console.log(apellido); // Pérez
                    console.log(edad); // 30
