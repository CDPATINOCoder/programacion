Imagenes En CSS

Las imagenes en CSS se pueden utilizar de varias formas, como fondo de un elemento, como imagen de un elemento, como imagen de fondo de la pagina, etc. En esta sección veremos algunas propiedades que nos permiten trabajar con imagenes en CSS.

Supongamos que estamos trabajando con una pagina web que tiene una sección de cards, y queremos que cada card tenga una imagen de fondo junto al titulo. Trabajaremos con este ejemplo donde un article contendrá la imagen y un h4 como titulo o referencia a la imagen.

object-fit

La propiedad object-fill nos permite definir como se ajusta la imagen al contenedor. Esta propiedad es muy util cuando trabajamos con imagenes como fondo de un elemento. A continuación veremos algunos valores que puede tomar esta propiedad.

fill

El valor fill ajusta la imagen al contenedor sin importar la relación de aspecto de la imagen. Esto puede hacer que la imagen se deforme.

Puesta de Sol

          
            img {
              width: 100%;
              height: auto;
            }

Primero reseteamos cualquier imagen de esta manera para que se ajuste de manera responsiva. 

          
            .card {
              width: 400px;
              height: 400px;
              background-color: aqua;
              border: medium solid black;
            }

Luego definimos el ancho y alto del contenedor de la imagen (la carta) y le damo su color de fondo. Esto lo hacemos simplemente para los ejemplos que veremos. 

          
            .img1 {
              height: 300px;
            object-fit: fill; /* valor por defecto */
            }

cover

El valor cover ajusta la imagen al contenedor sin deformarla, pero puede que la imagen no se muestre completamente.

Puesta de Sol

          
            .img2 {
              height: 300px;
              object-fit: cover;
            }

En este caso la imagen se ajusta al contenedor sin deformarse, pero puede que no se muestre completamente.

contain

El valor contain ajusta la imagen al contenedor sin deformarla y asegurandose de que la imagen se muestre completamente.

Puesta de Sol

          
            .img3 {
              height: 300px;
              object-fit: contain;
            }

En este caso la imagen se ajusta al contenedor sin deformarse y se asegura de que la imagen se muestre completamente. Esto puede hacer que la imagen no ocupe todo el contenedor.

none

El valor none no ajusta la imagen al contenedor, por lo que la imagen se muestra tal cual es.

Puesta de Sol

          
            .img4 {
              height: 300px;
              object-fit: none;
            }

En este caso la imagen no se ajusta al contenedor, por lo que se muestra tal cual es.

scale-down

El valor scale-down ajusta la imagen al contenedor sin deformarla y asegurandose de que la imagen se muestre completamente, pero solo si la imagen es mas grande que el contenedor.

Puesta de Sol

          
            .img5 {
              height: 300px;
              object-fit: scale-down;
            }

En este caso la imagen se ajusta al contenedor sin deformarse y se asegura de que la imagen se muestre completamente, pero solo si la imagen es mas grande que el contenedor.

object-position

La propiedad object-position nos permite definir la posición de la imagen en el contenedor. Esta propiedad es muy util cuando trabajamos con imagenes como fondo de un elemento. A continuación veremos algunos valores que puede tomar esta propiedad.

top

El valor top posiciona la imagen en la parte superior del contenedor.

Puesta de Sol

          
            .img6 {
              height: 300px;
              object-fit: contain;
              object-position: top;
            }

En este caso la imagen se posiciona en la parte superior del contenedor.

bottom

El valor bottom posiciona la imagen en la parte inferior del contenedor.

Puesta de Sol

          
            .img7 {
              height: 300px;
              object-fit: contain;
              object-position: bottom;
            }

En este caso la imagen se posiciona en la parte inferior del contenedor.

left

El valor left posiciona la imagen en la parte izquierda del contenedor.

Puesta de Sol

          
            .img8 {
              height: 300px;
              object-fit: contain;
              object-position: left;
            }

En este caso la imagen se posiciona en la parte izquierda del contenedor.

right

El valor right posiciona la imagen en la parte derecha del contenedor.

Puesta de Sol

          
            .img9 {
              height: 300px;
              object-fit: contain;
              object-position: right;
            }

En este caso la imagen se posiciona en la parte derecha del contenedor.

center

El valor center posiciona la imagen en el centro del contenedor.

Puesta de Sol

          
            .img10 {
              height: 300px;
              object-fit: contain;
              object-position: center;
            }

En este caso la imagen se posiciona en el centro del contenedor.

alejar del contenedor

Podemos alejar la imagen del contenedor utilizando valores negativos.

Puesta de Sol

          
            .img11 {
              height: 300px;
              object-fit: contain;
              object-position: -50px;
            }

En este caso la imagen se aleja del contenedor 50px en el lado derecho horizontal.
