Bordes y Outlines

Los bordes y contornos son una forma de darle estilo a los elementos de una página web. Los bordes se utilizan para rodear los elementos de la página, mientras que los contornos se utilizan para rodear los elementos de la página sin afectar el diseño de la página.

border-radius

La propiedad border-radius se utiliza para redondear las esquinas de un elemento. A continuación se muestra las distintas formas de redondear las esquinas de un elemento.

Todos las esquinas

Para redondear todos los bordes de un elemento se utiliza un solo valor con el short hand de la propiedad border-radius. 

          
            .border-radius-all {
              border-radius: 2rem;
            }

Diagonales

Para redondear las esquinas de un elemento de forma diagonal se utilizan dos valores con el short hand de la propiedad border-radius. 

          
            .border-radius-diagonal {
              border-radius: 2rem 4rem;
            }

Esquinas independientes

Para redondear las esquinas de un elemento de forma independiente se utilizan cuatro valores. 

          
            .border-radius-independent {
              border-radius: 2rem 4rem 6rem 8rem;
            }

No solamente podemos redondear de manera circular las esquinas de un elemento sino que también podemos redondear de manera elíptica. Cómo lo hacemos? Pues simplemente añadiendo un slash (/) entre los valores de la propiedad border-radius.

Todos las esquinas

Para redondear todos los bordes de un elemento de forma elíptica se utiliza un solo valor con el short hand de la propiedad border-radius. 

          
            .border-radius-all-elliptical {
              border-radius: 2rem / 4rem;
            }

Diagonales

Para redondear las esquinas de un elemento de forma diagonal de forma elíptica se utilizan dos valores con el short hand de la propiedad border-radius. 

          
            .border-radius-diagonal-elliptical {
              border-radius: 2rem 4rem / 6rem 8rem;
            }

Esquinas independientes

Para redondear las esquinas de un elemento de forma independiente de forma elíptica se utilizan cuatro valores. 

          
            .border-radius-independent-elliptical {
              border-radius: 2rem 4rem 6rem 8rem / 10rem 12rem 14rem 16rem;
            }

outline

La propiedad outline se utiliza para rodear los elementos de la página sin afectar el diseño de la página. A diferencia de la propiedad border, la propiedad outline no afecta el tamaño del elemento. Estos son las propiedades que se pueden utilizar con la propiedad outline.

  • outline-color:Define el color del outline o contorno
  • outline-style: Define el estilo del outline o contorno
    • none: No se muestra el outline o contorno
    • hidden: Se muestra el outline o contorno pero no se muestra
    • dotted: Se muestra el outline o contorno con puntos
    • dashed: Se muestra el outline o contorno con líneas discontinuas
    • solid: Se muestra el outline o contorno con líneas continuas
    • double: Se muestra el outline o contorno con dos líneas continuas
    • groove: Se muestra el outline o contorno con un efecto de hendidura
    • ridge: Se muestra el outline o contorno con un efecto de cresta
    • inset: Se muestra el outline o contorno con un efecto de hundimiento
    • outset: Se muestra el outline o contorno con un efecto de sobresalir
  • outline-width: Define el ancho del outline o contorno
  • outline-offset: Define la distancia entre el outline o contorno y el elemento 
        
          .outline {
            outline-color: blue;
            outline-style: solid;
            outline-width: 2px;
            outline-offset: 10px;
          }

Nota: Podemos utilizar simplemente el short hand ouline: width style color; para definir el outline de un elemento.
