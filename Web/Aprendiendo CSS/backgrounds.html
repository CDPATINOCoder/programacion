Estilos del Fondo

Los fondos en CSS son una forma de darle estilo a los elementos de una página web. Los fondos se utilizan para darle color o textura a los elementos de la página. En esta seccion veremos las propiedades que podemos utilizar para darle estilo a los fondos de los elementos de una página web.

background-color

La propiedad background-color se utiliza para darle color de fondo a un elemento. El valor de esta propiedad puede ser un color en formato hexadecimal, RGB o nombre de color. 

        
          .bg-color {
            background-color: green;
          }

background-image

La propiedad background-image se utiliza para darle una imagen de fondo a un elemento. El valor de esta propiedad puede ser la URL de la imagen que se quiere utilizar. 

        
          .bg-image{
            background-image: url("../assets/puesta-sol.jpg")
          }

background-size

La propiedad background-size se utiliza para especificar el tamaño de la imagen de fondo. Estos son los valores que puede tomar esta propiedad:

  • cover: La imagen se ajusta al tamaño del elemento y se recorta si es necesario.
  • contain: La imagen se ajusta al tamaño del elemento y se muestra completa. 
        
          .bg-size {
            background-image: url("../assets/puesta-sol.jpg");
            background-size: cover;
          }

background-repeat

La propiedad background-repeat se utiliza para especificar si la imagen de fondo se repite o no. Estos son los valores que puede tomar esta propiedad:

  • repeat: La imagen se repite en ambas direcciones (horizontal y vertical).
  • repeat-x: La imagen se repite solo en la dirección horizontal.
  • repeat-y: La imagen se repite solo en la dirección vertical.
  • no-repeat: La imagen no se repite.
        
          .bg-repeat {
            background-color: green;
            background-image: url("../assets/puesta-sol.jpg");
            background-size: cover;
            background-repeat: no-repeat;
          }

background-position

La propiedad background-position se utiliza para especificar la posición de la imagen de fondo. Estos son los valores que puede tomar esta propiedad:

  • top: La imagen se posiciona en la parte superior del elemento.
  • bottom: La imagen se posiciona en la parte inferior del elemento.
  • left: La imagen se posiciona en la parte izquierda del elemento.
  • right: La imagen se posiciona en la parte derecha del elemento.
  • center: La imagen se posiciona en el centro del elemento (horizontal y vertical).
  • x% y%: La imagen se posiciona en la posición especificada en porcentaje del elemento.
  • xpx ypx: La imagen se posiciona en la posición especificada en píxeles del elemento. 
        
          .bg-position {
            background-color: green;
            background-image: url("../assets/puesta-sol.jpg");
            background-size: cover;
            background-repeat: no-repeat;
            background-position: center;
          }

background-clip

La propiedad background-clip se utiliza para especificar la zona del elemento en la que se va a aplicar el fondo (imagen y color). Estos son los valores que puede tomar esta propiedad:

  • border-box: El fondo se aplica en toda la caja del elemento, incluyendo el borde.
  • padding-box: El fondo se aplica en la zona de relleno del elemento, excluyendo el borde.
  • content-box: El fondo se aplica en la zona de contenido del elemento, excluyendo el relleno y el borde. 
        
          .bg-clip {
            background-color: green;
            background-image: url("../assets/puesta-sol.jpg");
            background-size: cover;
            background-repeat: no-repeat;
            background-position: center;
            background-clip: padding-box;
          }

background-origin

La propiedad background-origin se utiliza para especificar el origen del fondo (imagen) en el elemento. Estos son los valores que puede tomar esta propiedad:

  • border-box: El fondo se aplica en toda la caja del elemento, incluyendo el borde.
  • padding-box: El fondo se aplica en la zona de relleno del elemento, excluyendo el borde.
  • content-box: El fondo se aplica en la zona de contenido del elemento, excluyendo el relleno y el borde. 
        
          .bg-origin {
            background-color: green;
            background-image: url("../assets/puesta-sol.jpg");
            background-size: cover;
            background-repeat: no-repeat;
            padding: 1rem;
            background-origin: content-box;
          }

background attachment

La propiedad background-attachment se utiliza para especificar si la imagen de fondo se desplaza con el contenido o se mantiene fija en su posición. Estos son los valores que puede tomar esta propiedad:

  • scroll: La imagen de fondo se desplaza con el contenido del elemento.
  • fixed: La imagen de fondo se mantiene fija en su posición mientras se desplaza el contenido del elemento.
  • local: La imagen de fondo se desplaza con el contenido del elemento, pero no se desplaza con el contenido de los elementos secundarios. 
        
          width: 100%;
          height: 100vh;
          background-color: green;
          background-image: url("../assets/puesta-sol.jpg");
          background-size: cover;
          background-repeat: no-repeat;
          padding: 1rem;
          background-attachment: fixed;

background

La propiedad background es una propiedad abreviada que se utiliza para especificar todas las propiedades de fondo en una sola declaración. Los valores que puede tomar esta propiedad son los mismos que los de las propiedades individuales. 

        
          .bg {
            width: 100%;
            height: 100vh;
            background: green url("../assets/puesta-sol.jpg") no-repeat center
              / cover fixed;
          }

Nota:: Con el shorthand background se pueden importar varias imagenes de fondo, separandolas por comas.
