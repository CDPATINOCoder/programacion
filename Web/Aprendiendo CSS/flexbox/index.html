Aprendiendo Flexbox

Flexbox es un modelo de diseño unidimensional que permite distribuir elementos en un contenedor y alinearlos de manera que ocupen el espacio disponible, con el objetivo de mejorar la distribución de los elementos en una interfaz y la alineación de los mismos.

En este documento aprenderemos a utilizar Flexbox para distribuir y alinear elementos en una interfaz web de manera sencilla y eficiente utilizando CSS. Veremos conceptos y ejemplos para cada propiedad de Flexbox y cómo aplicarlas en la práctica.

Antes de empezar a ver las propiedades que podemos utilizar con Flexbox, debemos de saber algunas cosas que vienen de manera predeterminada en flexbox:

  • Los elementos flexibles se distribuyen en una fila de manera predeterminada.
  • Los elementos flexibles se ajustan al tamaño del contenedor.
  • Los elementos flexibles se alinean en la dirección principal del contenedor.
  • Los elementos flexibles se alinean en el centro del contenedor.

Propiedad display: flex;

Para utilizar Flexbox en un contenedor, debemos definir la propiedad display: flex; en el contenedor. De esta manera, los elementos hijos del contenedor se convierten en elementos flexibles y pueden ser distribuidos y alineados de acuerdo a las propiedades de Flexbox. Trabajan en bloque.

Ejemplo:

            display: flex;
            }

En el ejemplo anterior, el contenedor .container1 tiene la propiedad display: flex; y los elementos hijos .item se distribuyen en una fila de manera predeterminada.

La propiedad display: inline-flex; es similar a la propiedad display: flex;, pero los elementos flexibles se distribuyen en una fila y se ajustan al tamaño del contenedor, sin ocupar todo el ancho disponible. Trabajan en linea.

Propiedad flex-grow

La propiedad flex-grow especifica cuánto debe crecer un elemento flexible en relación con los otros elementos flexibles cuando hay espacio adicional en el contenedor. El valor por defecto es flex-grow: 0;

Ejemplo:

item1
item2
item3
        
          .item-c33 {
            background-color: brown;
            border: thin solid black;
            width: 100px;
            flex-grow: 1;
          }
          .container33 {
            width: 100%;
            height: 50vh;
            background-color: aquamarine;
            display: flex;
          }

En el ejemplo anterior, los elementos flexibles tienen la propiedad flex-grow: 1; y crecen en relación con los otros elementos flexibles cuando hay espacio adicional en el contenedor.

Propiedad flex-shrink

La propiedad flex-shrink especifica cuánto debe encogerse un elemento flexible en relación con los otros elementos flexibles cuando no hay espacio suficiente en el contenedor. El valor por defecto es flex-shrink: 1;

Ejemplo:

item1
item2
item3
        
          .item-c34 {
            background-color: brown;
            border: thin solid black;
            width: 100px;
            flex-shrink: 1;
          }
          .item-c34:nth-child(2) {
            flex-shrink: 2;
          }
          .container34 {
            width: 100%;
            height: 50vh;
            background-color: aquamarine;
            display: flex;
          }

En el ejemplo anterior, los elementos flexibles tienen la propiedad flex-shrink: 1; y se encogen en relación con los otros elementos flexibles cuando no hay espacio suficiente en el contenedor. El segundo elemento tiene la propiedad flex-shrink: 2; y se encoge el doble que los otros elementos flexibles.

Nota: Si todos los elementos flexibles tienen flex-shrink: 0;, no se encogerán.

Propiedad flex-basis

La propiedad flex-basis especifica el tamaño inicial de un elemento flexible antes de que se distribuya el espacio adicional o se encoge. El valor por defecto es flex-basis: auto;

Ejemplo:

item1
item2
item3
        
          .item-c35 {
            background-color: brown;
            border: thin solid black;
            width: 100px;
            flex-basis: 200px;
          }
          .container35 {
            width: 100%;
            height: 50vh;
            background-color: aquamarine;
            display: flex;
          }

En el ejemplo anterior, los elementos flexibles tienen la propiedad flex-basis: 200px; y tienen un tamaño inicial de 200px antes de que se distribuya el espacio adicional o se encogen.

Si el flex direction es row, la propiedad flex-basis establece el ancho inicial del elemento flexible. Si el flex direction es column, la propiedad flex-basis establece la altura inicial del elemento flexible.

Shorthand flex

La propiedad flex es una propiedad abreviada que combina las propiedades flex-grow, flex-shrink y flex-basis en una sola propiedad.

Ejemplo:

item1
item2
item3
          
            .item-c36 {
              background-color: brown;
              border: thin solid black;
              flex: 1 1 200px;
            }
            .container36 {
              width: 100%;
              height: 50vh;
              background-color: aquamarine;
              display: flex;
            }

En el ejemplo anterior, los elementos flexibles tienen la propiedad flex: 1 1 200px; que es una propiedad abreviada de flex-grow: 1;, flex-shrink: 1; y flex-basis: 200px;.

Si se especifica un solo valor, este es el valor de flex-grow. Si se especifican dos valores, el primer valor es el valor de flex-grow y el segundo valor es el valor de flex-shrink. Si se especifican tres valores, el primer valor es el valor de flex-grow, el segundo valor es el valor de flex-shrink y el tercer valor es el valor de flex-basis.
