Colores, Unidades y Estilos En CSS

Colores

En CSS existen 4 sistemas para definir colores:

  • Palabras Clave: Son palabras reservadas que representan un color.
    • red
    • blue
    • green
    • yellow
    • black
    • white
    • gray
    • purple
    • orange
    • pink
    • ...
  • RGB: Es un sistema de colores que se basa en la mezcla de los colores rojo, verde y azul (Red, Green, Blue). Para definir la opacidad de un color se utiliza el cuarto valor (Alpha). se representa con el prefijo rgba.
    • rgb(255, 0, 0)
    • rgb(0, 255, 0)
    • rgb(0, 0, 255)
    • rgb(255, 255, 0)
    • rgb(0, 255, 255)
    • rgb(255, 0, 255)
    • rgb(255, 255, 255)
    • rgb(0, 0, 0)
    • rgba(255, 0, 0, 0.5)
  • HEX: Es un sistema de colores que se basa en la mezcla de los colores rojo, verde y azul (Red, Green, Blue). Se representa con el prefijo # seguido de 6 caracteres que representan los valores de los colores en hexadecimal. Para definir la opacidad de un color se utiliza el octavo caracter. Es decir se representa con 8 caracteres (6 para los colores y 2 para la opacidad).
    • #FF0000
    • #00FF00
    • #0000FF
    • #FFFF00
    • #00FFFF
    • #FF00FF
    • #FFFFFF
    • #000000
  • HSL: Es un sistema de colores que se basa en la mezcla de los colores tono, saturación y luminosidad (Hue, Saturation, Lightness). Para definir la opacidad de un color se utiliza el cuarto valor (Alpha). Se representa con el prefijo hsla.
    • hsl(0, 100%, 50%)
    • hsl(120, 100%, 50%)
    • hsl(240, 100%, 50%)
    • hsl(60, 100%, 50%)
    • hsl(180, 100%, 50%)
    • hsl(300, 100%, 50%)
    • hsl(0, 0%, 0%)
    • hsla(0, 100%, 50%, 0.5)

Nota: La opacidad de un color se define con un valor entre 0 y 1. Donde 0 es totalmente transparente y 1 es totalmente opaco.

Valores transparent y currentColor

En CSS existen dos valores especiales para definir colores:

transparent

El valor transparent se utiliza para definir un color transparente. Es decir, un color que no se visualiza. 

                    
                        background-color: transparent;

currentColor

El valor currentColor se utiliza para definir el color del texto. Es decir, el color que se define en la propiedad color. Este valor busca el ancestro más cercano que tenga definido un color y lo toma como referencia para definir el color del texto. 

                    
                        color: currentColor;

Propiedad Opacity

La propiedad opacity se utiliza para definir la opacidad de un elemento y su contenido. Esta propiedad acepta valores entre 0 y 1. Donde 0 es totalmente transparente y 1 es totalmente opaco. 

                
                    opacity: 0.5;

Nota: La propiedad opacity afecta a todos los elementos hijos del elemento al que se le aplica.
