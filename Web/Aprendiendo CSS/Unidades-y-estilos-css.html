Colores

En CSS existen 4 sistemas para definir colores:

  • Palabras Clave: Son palabras reservadas que representan un color.
    • red
    • blue
    • green
    • yellow
    • black
    • white
    • gray
    • purple
    • orange
    • pink
    • ...
  • RGB: Es un sistema de colores que se basa en la mezcla de los colores rojo, verde y azul (Red, Green, Blue). Para definir la opacidad de un color se utiliza el cuarto valor (Alpha). se representa con el prefijo rgba.
    • rgb(255, 0, 0)
    • rgb(0, 255, 0)
    • rgb(0, 0, 255)
    • rgb(255, 255, 0)
    • rgb(0, 255, 255)
    • rgb(255, 0, 255)
    • rgb(255, 255, 255)
    • rgb(0, 0, 0)
    • rgba(255, 0, 0, 0.5)
  • HEX: Es un sistema de colores que se basa en la mezcla de los colores rojo, verde y azul (Red, Green, Blue). Se representa con el prefijo # seguido de 6 caracteres que representan los valores de los colores en hexadecimal. Para definir la opacidad de un color se utiliza el octavo caracter. Es decir se representa con 8 caracteres (6 para los colores y 2 para la opacidad).
    • #FF0000
    • #00FF00
    • #0000FF
    • #FFFF00
    • #00FFFF
    • #FF00FF
    • #FFFFFF
    • #000000
  • HSL: Es un sistema de colores que se basa en la mezcla de los colores tono, saturación y luminosidad (Hue, Saturation, Lightness). Para definir la opacidad de un color se utiliza el cuarto valor (Alpha). Se representa con el prefijo hsla.
    • hsl(0, 100%, 50%)
    • hsl(120, 100%, 50%)
    • hsl(240, 100%, 50%)
    • hsl(60, 100%, 50%)
    • hsl(180, 100%, 50%)
    • hsl(300, 100%, 50%)
    • hsl(0, 0%, 0%)
    • hsla(0, 100%, 50%, 0.5)

Nota: La opacidad de un color se define con un valor entre 0 y 1. Donde 0 es totalmente transparente y 1 es totalmente opaco.

Valores transparent y currentColor

En CSS existen dos valores especiales para definir colores:

transparent

El valor transparent se utiliza para definir un color transparente. Es decir, un color que no se visualiza. 

                    
                        background-color: transparent;

currentColor

El valor currentColor se utiliza para definir el color del texto. Es decir, el color que se define en la propiedad color. Este valor busca el ancestro más cercano que tenga definido un color y lo toma como referencia para definir el color del texto. 

                    
                        color: currentColor;

Propiedad Opacity

La propiedad opacity se utiliza para definir la opacidad de un elemento y su contenido. Esta propiedad acepta valores entre 0 y 1. Donde 0 es totalmente transparente y 1 es totalmente opaco. 

                
                    opacity: 0.5;

Nota: La propiedad opacity afecta a todos los elementos hijos del elemento al que se le aplica.

Unidades de Medidas

En CSS existen diferentes unidades de medida para definir tamaños y distancias, los cuales se clasifican en absolutos y relativos.

Absolutas

Las unidades absolutas son fijas y no cambian su tamaño. En esta sección veremos una unidad absoluta.

pixeles

La unidad de medida px se utiliza para definir tamaños y distancias en pixeles. Un pixel es la unidad mínima de una pantalla. 

 
                        
                            width: 100px;

Relativas

Las unidades relativas son dinámicas y cambian su tamaño en función de un valor de referencia las cuales pueden ser el tamaño de la fuente, el tamaño de la pantalla, el tamaño del contenedor, entre otros. En esta sección veremos algunas unidades relativas.

em

La unidad de medida em se utiliza para definir tamaños y distancias en función del tamaño de la fuente del elemento padre. Supongamos que el tamaño de la fuente del elemento padre es de 16px, si definimos un tamaño de 2em, el tamaño del elemento será de 32px. 

                        
                            font-size: 2em; // 32px

rem

La unidad de medida rem se utiliza para definir tamaños y distancias en función del tamaño de la fuente del elemento raíz (root). Por defecto el tamaño de la fuente del elemento raíz es de 16px. 

                        
                            font-size: 2rem; // 32px

ex

La unidad de medida ex se utiliza para definir tamaños y distancias en función de la altura de la fuente del elemento padre. Supongamos que la altura de la fuente del elemento padre es de 16px, si definimos un tamaño de 2ex, el tamaño del elemento será de 32px. 

                        
                            font-size: 2ex; // 32px

Nota: La altura de la fuente es la distancia entre la línea base y la línea superior de la fuente. Generalmente es la altura de la letra "x".

ch

La unidad de medida ch se utiliza para definir tamaños y distancias en función de la cantidad de caracteres "0" que caben en el espacio de la fuente del elemento padre. Supongamos que la cantidad de caracteres "0" que caben en el espacio de la fuente del elemento padre es de 10 caracteres, si definimos un ancho de 10ch el tamaño se ajusta de acuerdo a la cantidad de caracteres "0" que caben en el espacio de la fuente del elemento padre. 

                        
                            width: 10ch;

vw

La unidad de medida vw se utiliza para definir tamaños y distancias en función del ancho de la pantalla. Un valor de 1vw es igual al 1% del ancho de la pantalla. 

                        
                            width: 100vw; // 100% del ancho de la pantalla

vh

La unidad de medida vh se utiliza para definir tamaños y distancias en función de la altura de la pantalla. Un valor de 1vh es igual al 1% de la altura de la pantalla. 

                        
                            height: 100vh; // 100% de la altura de la pantalla

Nota: Las unidades vw y vh son útiles para definir tamaños y distancias en función del tamaño de la pantalla.

vmin

La unidad de medida vmin se utiliza para definir tamaños y distancias en función del tamaño más pequeño entre el ancho y la altura de la pantalla. Un valor de 1vmin es igual al 1% del tamaño más pequeño entre el ancho y la altura de la pantalla. 

                        
                            width: 100vmin; // 100% del tamaño más pequeño entre el ancho y la altura de la pantalla, si el ancho es menor que la altura, el tamaño será del 100% del ancho, si la altura es menor que el ancho, el tamaño será del 100% de la altura.

vmax

La unidad de medida vmax se utiliza para definir tamaños y distancias en función del tamaño más grande entre el ancho y la altura de la pantalla. Un valor de 1vmax es igual al 1% del tamaño más grande entre el ancho y la altura de la pantalla. 

                        
                            width: 100vmax; // 100% del tamaño más grande entre el ancho y la altura de la pantalla, si el ancho es mayor que la altura, el tamaño será del 100% del ancho, si la altura es mayor que el ancho, el tamaño será del 100% de la altura.
