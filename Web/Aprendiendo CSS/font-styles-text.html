font-face

La regla @font-face nos permite definir una fuente personalizada que podemos usar en nuestras páginas web. Para definir una fuente personalizada, debemos especificar el nombre de la fuente, la ruta de la fuente y el formato de la fuente. Por ejemplo, si queremos definir una fuente personalizada llamada "Pecita.woff" que se encuentra en el archivo "myfont.woff" en la carpeta "assets" y que es de formato WOFF, debemos especificar: 

          
            @font-face {
              font-family: 'Pecita', sans-serif;
              src: url("assets/Pecita.woff") format("woff");
            }

font-display

La propiedad font-display nos permite especificar cómo se debe mostrar una fuente personalizada en nuestras páginas web. Para especificar cómo se debe mostrar una fuente personalizada, debemos especificar uno de los siguientes valores:

  • auto: Es el valor por defecto. Indica que el navegador debe elegir cómo mostrar la fuente personalizada.
  • block: Indica que el navegador debe bloquear la carga de la fuente personalizada hasta que la fuente esté disponible.
  • swap: Indica que el navegador debe mostrar una fuente de respaldo hasta que la fuente personalizada esté disponible.
  • fallback: Indica que el navegador debe mostrar una fuente de respaldo hasta que la fuente personalizada esté disponible. Si la fuente personalizada no está disponible, el navegador debe mostrar una fuente de respaldo.
  • optional: Indica que el navegador debe mostrar una fuente de respaldo hasta que la fuente personalizada esté disponible. Si la fuente personalizada no está disponible, el navegador no debe mostrar la fuente personalizada. 
          
            @font-face {
              font-family: 'Pecita', sans-serif;
              src: url("assets/Pecita.woff") format("woff");
              font-display: swap; // Indica que el navegador debe mostrar una fuente de respaldo hasta que la fuente personalizada esté disponible.
            }

Fuentes de Google

Google Fonts es una colección de fuentes web gratuitas que podemos usar en nuestras páginas web. Para usar una fuente de Google Fonts en nuestras páginas web, debemos seguir los siguientes pasos:

Paso 1: Seleccionar una Fuente

En la página de Google Fonts, debemos seleccionar la fuente que queremos usar en nuestras páginas web. Para seleccionar una fuente, debemos hacer clic en el botón "Select this style" que se encuentra debajo de la fuente que queremos usar.

Paso 2: Obtener el Código de la Fuente

Una vez que hayamos seleccionado la fuente que queremos usar en nuestras páginas web, debemos obtener el código de la fuente. Para obtener el código de la fuente, debemos hacer clic en el botón "Embed" que se encuentra en la parte superior derecha de la página de Google Fonts.

Paso 3: Agregar el Código de la Fuente a nuestras Páginas Web

Una vez que hayamos obtenido el código de la fuente, debemos agregar el código de la fuente a nuestras páginas web. Para agregar el código de la fuente a nuestras páginas web, debemos copiar y pegar el código de la fuente en el archivo HTML de nuestras páginas web.

Estilos de Texto

En CSS podemos cambiar el estilo de los textos de nuestras páginas web. En esta sección veremos algunas propiedades de CSS que nos permiten cambiar el estilo de los textos.

text-align

La propiedad text-align nos permite alinear el texto de un elemento HTML. Para alinear el texto de un elemento HTML, debemos especificar la alineación del texto que queremos usar. Por ejemplo, si queremos alinear el texto de un elemento HTML al centro, debemos especificar text-align: center;. Estos son algunos valores que pueden tomar la propiedad text-align:

  • left: Indica que el texto debe estar alineado a la izquierda.
  • right: Indica que el texto debe estar alineado a la derecha.
  • center: Indica que el texto debe estar alineado al centro.
  • justify: Indica que el texto debe estar justificado. Esto significa que el texto debe estar alineado a ambos lados.

Los iconos tipográficos son iconos que se pueden usar en lugar de texto en nuestras páginas web. En esta sección veremos cómo usar iconos tipográficos de algunas de las cdns más populares.

Google Material Icons

Google Material Icons es una colección de iconos tipográficos que podemos usar en nuestras páginas web. Para usar un icono tipográfico de Google Material Icons en nuestras páginas web, debemos seguir los siguientes pasos:

Paso 1:

En la página de Google Material Icons, debemos seleccionar el icono que queremos usar en nuestras páginas web. Para seleccionar un icono, debemos hacer clic en el icono que queremos usar.

Paso 2:

Una vez que hayamos seleccionado el icono que queremos usar en nuestras páginas web debemos de llamar a la cdn de Google Material Icons en el archivo HTML de nuestras páginas web. 

            
              <head>
              <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Icons+Outlined:opsz,wght,FILL,GRAD@24,400,0,0"/>
              </head>

Paso 3:

Una vez que hayamos llamado a la cdn de Google Material Icons en el archivo HTML de nuestras páginas web, debemos de copiar el código proporcionado por la página web y pegarlo en el archivo HTML de nuestras páginas web. 

            
              <body>
              <i class="material-icons">face</i>
              </body>
