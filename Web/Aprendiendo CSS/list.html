Listas En CSS

En esta sección estaremos viendo algunas de las propiedades y valores que podemos aplicar a las listas en CSS.

list-style-type

Esta propiedad nos permite cambiar el tipo de marcador que se muestra en la lista. Los valores que podemos utilizar son:

  • disc: Círculo
  • circle: Círculo vacío
  • square: Cuadrado
  • decimal: Números decimales
  • decimal-leading-zero: Números decimales con cero al inicio
  • lower-roman: Números romanos en minúsculas
  • upper-roman: Números romanos en mayúsculas
  • lower-alpha: Letras en minúsculas
  • upper-alpha: Letras en mayúsculas
  • none: Sin marcador

Ejemplo:

  • primavera
  • verano
  • otoño
  • invierno
        
          .list1 {
            list-style-type: square;
          }

En el ejemplo anterior hemos cambiado el tipo de marcador de la lista a cuadrado.

list-style-image

Esta propiedad nos permite utilizar una imagen como marcador de la lista. Para ello, debemos especificar la ruta de la imagen que queremos utilizar.

Para utilizar esta propiedad necesitamos que la imagen sea lo suficientemente pequeña para que no se vea distorsionada. Como de unos 16px x 16px.

  • primavera
  • verano
  • otoño
  • invierno
        
          .list2 {
            list-style-image: url('../assests/icon.png')
          }

como se aprecia en el ejemplo la imagen se deformó completamente, la mejor manera de hacer marcadores con imagenes es trabajando con los elementos li de la lista, utilizando la pseudoclase before.

Ejemplo:

  • primavera
  • verano
  • otoño
  • invierno
        
          .list3 li::before {
          width: 1rem;
          height: 1rem;
          display: inline-block;
          content: "";
          background-image: url("../assets/avatar.png");
          background-size: contain;
          background-position: center;
          background-repeat: no-repeat;
          }

          .list3 {
          list-style-type: none;
          }

Lo que estamos haciendo aquí es agregar un marcador a cada elemento li de la lista, utilizando la pseudoclase before. En este caso, estamos utilizando una imagen como marcador, pero también podríamos utilizar un icono de una fuente como FontAwesome.

list-style-position

Esta propiedad nos permite cambiar la posición del marcador de la lista. Los valores que podemos utilizar son:

  • inside: Dentro del contenedor del elemento li
  • outside: Fuera del contenedor del elemento li

Ejemplo:

  • primavera
  • verano
  • otoño
  • invierno
        
          .list4 {
            list-style-position: inside;
          }

En el ejemplo anterior hemos cambiado la posición del marcador de la lista a dentro del contenedor del elemento li.

list-style

Esta propiedad nos permite establecer todas las propiedades de la lista en una sola declaración. Los valores que podemos utilizar son:

  • list-style-type
  • list-style-position
  • list-style-image

Ejemplo:

  • primavera
  • verano
  • otoño
  • invierno
          
            .list5 {
              list-style: square inside 
            }

En el ejemplo anterior hemos cambiado el tipo de marcador de la lista a cuadrado y la posición del marcador a dentro del contenedor del elemento li.

Listas y Textos en Columnas

En CSS podemos dividir el contenido de un elemento en columnas.

column-count

Esta propiedad nos permite dividir el contenido de un elemento en un número determinado de columnas.

Ejemplo:

  • primavera
  • verano
  • otoño
  • invierno

  • verano

  • invierno
        
          .list6 {
            column-count: 3;
          }

column-gap

Esta propiedad nos permite establecer el espacio entre las columnas.

Ejemplo:

  • primavera
  • verano
  • otoño
  • invierno

  • verano

          .list7 {
            column-count: 3;
            column-gap: 1rem;
          }

column-rule

Esta propiedad nos permite establecer una línea entre las columnas.

Ejemplo:

  • primavera
  • verano

  • invierno

column-width

Esta propiedad nos permite establecer el ancho máximo de las columnas. Si utilizamos esta propiedad, no podemos utilizar column-count.
