Aprendiendo CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para describir la presentación de un documento escrito en HTML o XML. Mientras que HTML estructura el contenido y define los elementos que componen una página web, CSS se encarga de la apariencia visual de esos elementos, como colores, fuentes, espaciamiento, y disposición en la pantalla.

Algo muy importante que debemos de saber es que una regla CSS consta de dos partes el "selector" y el "bloque de declaraciones".

🔼

Selectores Básicos

Los selectores básicos son los que se utilizan para seleccionar elementos HTML. Estos son 3 de los selectores más comunes:

Selector De Tipo

Este selector se utiliza para seleccionar todos los elementos de un tipo específico. Por ejemplo, si queremos seleccionar todos los elementos de tipo <p> en nuestro documento, podemos utilizar el siguiente selector: 

                            
                                p {
                                    color: red;
                                }

Selector De Clase

Este selector se utiliza para seleccionar todos los elementos que tienen una clase específica. Por ejemplo, si queremos seleccionar todos los elementos con la clase "texto-rojo" en nuestro documento, podemos utilizar el siguiente selector: 

                            
                                .texto-rojo {
                                    color: red;
                                }

Selector De ID

Este selector se utiliza para seleccionar un único elemento que tiene un ID específico. Por ejemplo, si queremos seleccionar el elemento con el ID "titulo" en nuestro documento, podemos utilizar el siguiente selector: 

                                
                                    #titulo {
                                        font-size: 24px;
                                    }

Selectores Avanzados

Los selectores avanzados son los que se utilizan para seleccionar elementos HTML de manera más específica. Algunos de los selectores avanzados más comunes son:

Selector De Hijos Directos

Este selector se utiliza para seleccionar los elementos que son hijos directos de otro elemento. Por ejemplo, si queremos seleccionar todos los elementos <p> que son hijos directos de un elemento <div>, podemos utilizar el siguiente selector: 

                        
                            div > p {
                                color: blue;
                            }

Si queremos seleccionar todos los elementos que son hijos directos de la lista <ul>, podemos utilizar el siguiente selector: 

                            
                                .selector-hijo-directo > li {
                                    color: azure;
                                    background-color: red;
                                }

Para decir que un elemento o varios elementos son hijos directos de un elemento este debe de estar inmediatamente después del elemento padre.

Selector De Hijos Descendientes

Este selector se utiliza para seleccionar los elementos que son descendientes de otro elemento. Por ejemplo, si queremos seleccionar todos los elementos <p> que son descendientes de un elemento <div>, podemos utilizar el siguiente selector: 

                        
                            div p {
                                color: green;
                            }

Otro ejemplo tenemos esta lista:

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4

Si queremos seleccionar todos los elementos que son descendientes de la lista <ul>, podemos utilizar el siguiente selector: 

                            
                                .selector-hijo-descendiente li {
                                    color: yellow;
                                    background-color: blue;
                                }

Para decir que un elemento o varios elementos son descendientes de un elemento este puede estar en cualquier parte del documento.

Selector De Hermanos General

Este selector se utiliza para seleccionar los elementos que son hermanos de otro elemento. Por ejemplo tenemos esta lista:

Si queremos seleccionar todos los elementos que son hermanos del elemento 2 <li>, podemos utilizar el siguiente selector: 

                        
                            .selector-hermano-general ~ li {
                                color: aqua;
                            }

Para decir que un elemento o varios elementos son hermanos de un elemento este debe de estar después del elemento padre.

Selector De Hermanos Adyacentes

Este selector se utiliza para seleccionar los elementos que son hermanos adyacentes de otro elemento. Por ejemplo tenemos esta lista:

  • Elemento 3

Si queremos seleccionar todos los elementos que son hermanos adyacentes del elemento 2 <li>, podemos utilizar el siguiente selector: 

                            
                                .selector-hermano-adyacente + li {
                                    color: fuchsia;
                                }

Para decir que un elemento o varios elementos son hermanos adyacentes de un elemento este debe de estar inmediatamente después del elemento padre.

Selector De Atributos

Este selector se utiliza para seleccionar los elementos que tienen un atributo específico. Por ejemplo, si queremos seleccionar todos los elementos con el atributo "href" en nuestro documento, podemos utilizar el siguiente selector: 

                            
                                [href] {
                                    color: purple;
                                }

Otro ejemplo, tenemos este menú de navegación:

  • Home
  • About
  • Services
  • Contact
  • Regresar al temario

Si queremos seleccionar todos los elementos que tengan el atributo "target" en nuestro menú de navegación, podemos utilizar el siguiente selector: 

                            
                                .selectores-atributos a[target] {
                                    color: orange;
                                }

Si queremos seleccionar todos los elementos que tengan el atributo "target" con el valor "_blank" en nuestro menú de navegación, podemos utilizar el siguiente selector: 

                            
                                .selectores-atributos a[target="_blank"] {
                                    color: pink;
                                }

Si queremos seleccionar todos los elementos que tengan el atributo href con la palabra "contact" en nuestro menú de navegación, podemos utilizar el siguiente selector: 

                            
                                .selectores-atributos a[href*="contact"] {
                                    color: brown;
                                }

Si queremos todos los elementos que tengan el atributo href y que empiece con la palabra http en nuestro menú de navegación, podemos utilizar el siguiente selector: 

                            
                                .selectores-atributos a[href^="http"] {
                                    color: gray;
                                }

Si queremos todos los elementos que tengan el atributo href y que termine con la palabra LA en nuestro menú de navegación, podemos utilizar el siguiente selector: 

                                
                                    .selectores-atributos a[href$="LA"] {
                                        color: black;
                                    }

Supongamos que en esta lista de navegación vista anteriormente, el hipervinculo llamado services tiene atributos de clase llamados class="servicios servicios-activos servicios-terminados" para aplicar estilos a un atributo que como valor contiene espacios usamos el comodin ~ ahora seleccionamos todos los elementos que tengan el atributo class con el valor servicios-activos en nuestro menú de navegación, podemos utilizar el siguiente selector: 

                                
                                    .selectores-atributos a[class~="servicios-activos"] {
                                        color: white;
                                        background-color: black;
                                    }

Si queremos buscar una palabra en concreto en un atributo que tenga varios valores y que la palabra buscada esté antes de un guion medio usamos el comodin | ahora seleccionamos todos los elementos que tengan el atributo class con el valor sobre en nuestro menú de navegación, podemos utilizar el siguiente selector: 

                                
                                    .selectores-atributos a[class|="sobre"] {
                                        color: white;
                                        background-color: black;
                                    }

Selector Universal

Este selector se utiliza para seleccionar todos los elementos en un documento. Por ejemplo, si queremos aplicar un estilo de fuente a todos los elementos en nuestro documento, podemos utilizar el siguiente selector: 

                        
                            * {
                                font-family: Arial, sans-serif;
                            }

Otro ejemplo, tenemos esta lista donde la ul tiene una clase llamada selector-universal:

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4

Si queremos aplicar un estilo de fuente a todos los elementos de la lista <ul>, podemos utilizar el siguiente selector: 

                        
                            .selector-universal * {
                                font-family: Arial, sans-serif;
                            }

Recuerda que el selector universal se utiliza para seleccionar todos los elementos en un documento, por lo que es importante tener cuidado al utilizarlo, ya que puede afectar a todos los elementos en la página.

Pseudoclases

Las pseudoclases son palabras clave que se añaden a los selectores para especificar un estado especial del elemento seleccionado. Algunas de las pseudoclases más comunes son:

Pseudoclase required



Si queremos aplicar un estilo a los campos de texto que son requeridos en nuestro formulario, podemos utilizar el siguiente selector: 

                            
                                .form-pseudoclases1 *:required {
                                    border: 2px solid red;
                                }

Pseudoclase Valid



Si queremos aplicar un estilo a los campos de texto que son válidos en nuestro formulario, podemos utilizar el siguiente selector:

Pseudoclase Invalid



Si queremos aplicar un estilo a los campos de texto que son inválidos en nuestro formulario, podemos utilizar el siguiente selector:

Pseudoclase Focus



Si queremos aplicar un estilo a los campos de texto que tienen el foco en nuestro formulario, podemos utilizar el siguiente selector: 

                            
                                .form-pseudoclases4 *:focus {

Pseudoclase Disabled



Si queremos aplicar un estilo a los campos de texto que están deshabilitados en nuestro formulario, podemos utilizar el siguiente selector:

                                    color: white;
                                    background-color: black;
                                }

Pseudoelementos

Los pseudoelementos son palabras clave que se añaden a los selectores para especificar una parte especial del elemento seleccionado. A continuación vermeos algunas de ellas.

Before y After

Los pseudoelementos ::before y ::after se utilizan para insertar contenido antes y después del contenido de un elemento, respectivamente. Supongamos que tenemos un producto que tiene nombre, texto y precio.

Cámara Sony alpha 7 III 300

Si queremos añadir un guion antes del nombre del producto, el modelo en negrita y un signo de dolar después del precio, podemos utilizar los pseudoelementos ::before y ::after de la siguiente manera: 

                            
                                .product-name::before {
                                    content: "-";
                                }
                                .product-model::before {
                                    content: "Modelo: ";
                                    font-weight: bold;
                                }
                                .product-price::after {
                                    content: "$";

First-letter, First-line y selection

Los pseudoelementos ::first-letter y ::first-line se utilizan para aplicar estilos al primer carácter y a la primera línea de un elemento, respectivamente. Por otro lado, el pseudoelemento ::selection se utiliza para aplicar estilos al texto seleccionado por el usuario. Usaremos un parrafo de ejemplo.

Cuando queremos aplicar estilos a varios elementos con las mismas propiedades, podemos agrupar los selectores en una sola regla CSS. supongamos que tenemos un formulario de comentarios siguiente:

Si queremos aplicar algunos estilos a los campos de texto y al área de texto del formulario anterior, podemos agrupar los selectores en una sola regla CSS de la siguiente manera:

El Algoritmo De CSS

Cuando un navegador renderiza una página web, sigue un algoritmo para determinar qué estilos aplicar a cada elemento. Este algoritmo podríamos decir que se subdivide en 3 partes:

La Cascada

La cascada es el orden en el que el navegador aplica los estilos. Y toma en cuenta lo siguiente en este orden:

  1. El origen del códido:
    • El user agent (los estilos aplicados por defecto del navegador)
    • Personalizaciones del usuario
    • Los estilos del autor (nuestras modificaciones)
  2. La especificidad del selector
  3. Orden de aparición de los estilos en el código

En simples palabras cuando tienes una hoja de estilos el codigo se lee desde arriba hasta abajo y de izquierda a derecha, si tienes un estilo que se repite el que se lee al final es el que se aplica, ya que se sobreescribe.

La Especificidad

La especificidad es la forma en la que el navegador determina qué regla CSS tiene prioridad sobre otra. La especificidad se mide en cuatro valores:

  1. Etiquetas y pseudoelementos:
    • 1 punto (0,0,0,1)
  2. Clases, atributos y pseudoclases:
    • 10 puntos (0,0,1,0)
  3. IDs:
    • 100 puntos (0,1,0,0)
  4. Estilos en línea:
    • 1000 puntos (1,0,0.0)
  5. !important:
    • Rompe la especificidad, no importan los demás valores, se aplica el estilo con !important.

La Herencia

La herencia es el proceso por el cual los estilos de un elemento ancestro se aplican a sus elementos descendientes si estos no tienen estilos propios. Por ejemplo, si aplicamos un estilo de fuente a un elemento <p>, todos los elementos <p> dentro de ese elemento heredarán ese estilo de fuente. Veamos este párrafo de ejemplo.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Ratione excepturi repudiandae ipsum praesentium? Totam quasi suscipit nesciunt officiis eos repellendus sequi consequuntur quos quam sapiente provident, eveniet quae! Asperiores, dolorem!

Si queremos aplicar un estilo de fuente al párrafo anterior, podemos utilizar el siguiente selector: 

                            
                                .herencia {
                                    font-family: Arial, sans-serif;
                                    color: green;
                                }

Todos los elementos <strong> y <i> dentro del párrafo heredarán el estilo de fuente del párrafo. Pero no pueden heredar cosas como el color de fondo o el color de texto. Si queremos que no hereden el estilo de los elementos ancestros usamos otro valor de la propiedad font-family. 

                            
                                .herencia {
                                    font-family: Arial, sans-serif;
                                    color: green;
                                }
                                .herencia strong, .herencia i {
                                    font-family: sans-serif;
                                }

Si queremos que un elemento herede los estilos de su ancestro usamos la propiedad inherit. 

                            
                                .herencia strong {
                                    color: inherit;
                                }

Reseteo y Normalización de Estilos

Cuando creamos una página web, los navegadores aplican estilos por defecto a los elementos HTML. Estos estilos pueden variar de un navegador a otro, lo que puede causar problemas de compatibilidad. Para evitar estos problemas, es recomendable utilizar un reseteo o normalización de estilos. A continuación veremos en qué consisten estos dos métodos.

Reseteo de Estilos:

El reseteo de estilos es una técnica que consiste en eliminar los estilos por defecto de los elementos HTML para empezar desde cero. De esta manera, podemos asegurarnos de que todos los elementos se vean igual en todos los navegadores. A continuación se muestra un ejemplo de un reseteo de estilos: 

                        
                            * {
                                margin: 0;
                                padding: 0;
                                box-sizing: border-box;
                            }

Normalización de Estilos:

La normalización de estilos es una técnica que consiste en establecer estilos por defecto para los elementos HTML para que se vean igual en todos los navegadores. Lo recomendable es importar un archivo CSS que contenga los estilos normalizados como normalize.css o reset.css. A continuación se muestra un ejemplo de un archivo CSS que contiene estilos normalizados: 

                        
                            /* normalize.css v8.0.1 | MIT License | github.com/necolas/normalize.css */
                            /* HTML5 display definitions
                            ========================================================================== */
                            /**
                            * Correct `block` display not defined for any HTML5 element in IE 8/9.
                            * Correct `block` display not defined for `details` or `summary` in IE 10/11 and Firefox.
                            * Correct `block` display not defined for `main` in IE 11.
                            */
                            article,
                            aside,
                            details,
                            figcaption,
                            figure,
                            footer,
                            header,
                            hgroup,
                            main,
                            menu,
                            nav,
                            section,
                            summary {
                                display: block;
                            }
                            /**
                            * 1. Correct `inline-block` display not defined in IE 8/9.
                            * 2. Normalize vertical alignment of `progress` in Chrome, Firefox, and Opera.
                            */
                            audio,
                            canvas,
                            progress,
                            video {
                                display: inline-block;
                                vertical-align: baseline;
                            }
                            /**
                            * Prevent modern browsers from displaying `audio` without controls.
                            * Remove excess height in iOS 5 devices.
                            */
                            audio:not([controls]) {
                                display: none;
                                height: 0;
                            }
                            /**
                            * Address `[hidden]` styling not present in IE 8/9/10.
                            * Hide the `template` element in IE 8/9/11, Safari, and Firefox < 22.
                            */
                            [hidden],
                            template {
                                display: none;
                            }
                            /* Links
                            ========================================================================== */
                            /**
                            * Remove the gray background color from active links in IE 10.
                            */
                            a {
                                background-color: transparent;
                            }
                            /**
                            * Improve readability when focused and also mouse hovered in all browsers.
                            */
                            a:active,
                            a:hover {
                                outline-width: 0;
                            }

Prefijos de los Navegadores

Los prefijos de los navegadores son una técnica que se utiliza para aplicar estilos específicos a un navegador en particular. Esto se hace añadiendo un prefijo al nombre de la propiedad CSS. A continuación veremos algunos de los prefijos más comunes: 

                        
                            -webkit- (Google Chrome, Safari, Opera)
                            -moz- (Firefox)
                            -o- (Opera)
                            -ms- (Internet Explorer)

Por ejemplo, si queremos aplicar un estilo de borde redondeado a un elemento en Google Chrome, Safari y Opera, podemos utilizar el siguiente selector: 

                        
                            .elemento {
                                -webkit-border-radius: 10px;
                                border-radius: 10px;
                            }

Si queremos aplicar un estilo de borde redondeado a un elemento en Firefox, podemos utilizar el siguiente selector: 

                        
                            .elemento {
                                -moz-border-radius: 10px;
                                border-radius: 10px;
                            }

Si queremos aplicar un estilo de borde redondeado a un elemento en Internet Explorer, podemos utilizar el siguiente selector: 

                        
                            .elemento {
                                -ms-border-radius: 10px;
                                border-radius: 10px;
                            }

Es importante tener en cuenta que los prefijos de los navegadores se utilizan para aplicar estilos específicos a un navegador en particular, por lo que es recomendable utilizarlos con moderación y comprobar la compatibilidad con los navegadores antes de utilizarlos.

El relleno (padding) es el espacio entre el contenido y el borde de un elemento HTML. El margen (margin) es el espacio entre el borde de un elemento y los elementos adyacentes. Cajas de Linea vs Cajas de Bloque

En CSS, los elementos HTML se dividen en dos tipos: cajas de línea (inline) y cajas de bloque (block). Las cajas de línea son elementos que se muestran en la misma línea que el contenido que las rodea, mientras que las cajas de bloque son elementos que ocupan toda la anchura disponible y se muestran en una nueva línea. A continuación veremos la diferencia entre las cajas de línea y las cajas de bloque. Usaremos el archivo HTML box-model nuevamente para ejemplificarlo. ir al archivo

Cajas de Linea

Las cajas de línea son elementos que se muestran en la misma línea que el contenido que las rodea. Algunos ejemplos de cajas de línea son los elementos <span>, <a>, <strong>, <em>, etc.

Las cajas de linea al aplicar propiedades como width y height no se ven afectadas, ya que estas propiedades no se aplican a las cajas de linea. Otra cosa es que las cajas de linea no pueden tener padding ni margin en los lados superior e inferior. Estos no generan saltos de Linea, solo empujan el contenido.

Cajas de Bloque

Las cajas de bloque son elementos que ocupan toda la anchura disponible y se muestran en una nueva línea. Algunos ejemplos de cajas de bloque son los elementos <div>, <p>, <h1>, <ul>, <ol>, etc.

Las cajas de bloque al aplicar propiedades como width y height se ven afectadas, ya que estas propiedades se aplican a las cajas de bloque. Otra cosa es que las cajas de bloque pueden tener padding y margin en los lados superior e inferior. Estos generan saltos de Linea.

Propiedad Display

La propiedad display se utiliza para especificar cómo se mostrará un elemento HTML en la página. Hay varios valores que se pueden utilizar con la propiedad display, cada uno con un comportamiento diferente. A continuación veremos algunos de los valores más comunes de la propiedad display.

Inline

Permite cambiar una caja de bloque a una caja de línea. Esto sirve para que los elementos se muestren en la misma línea que el contenido que los rodea. 

                        
                            .caja-uno {
                                display: inline;
                            }

Block

Permite cambiar una caja de línea a una caja de bloque. Esto sirve para que los elementos ocupen toda la anchura disponible y se muestren en una nueva línea. 

                        
                            .caja-dos {
                                display: block;
                            }

Inline block

Permite cambiar una caja de línea o una caja de bloque a una caja de línea en bloque. Esto sirve para poder hacer una combinación de las propiedades de los dos tipos de cajas ya vistos, es decir, que los elementos se muestren en la misma línea que el contenido que los rodea y ocupen toda la anchura disponible. El width y el height se pueden aplicar a los elementos con display: inline-block. 

                        
                            .caja-tres {
                                display: inline-block;
                            }

None

Permite ocultar un elemento HTML de la página. Esto sirve para que los elementos no se muestren en la página. 

                        
                            .caja-cuatro {
                                display: none;
                            }

Propiedad Visibility

La propiedad visibility se utiliza para especificar si un elemento HTML es visible o no. Hay dos valores que se pueden utilizar con la propiedad visibility: visible y hidden. A continuación veremos cómo se utilizan estos valores en la propiedad visibility.

visible

Permite que un elemento HTML sea visible en la página. Esto sirve para que los elementos se muestren en la página. 

                        
                            .caja-uno {

hidden

Permite ocultar un elemento HTML de la página. Esto sirve para que los elementos no se muestren en la página. A diferencia de none en display, hidden en visibility oculta el elemento pero deja su espacio en la página. 

                        
                            .caja-dos {

Propiedad Overflow

La propiedad overflow se utiliza para especificar qué hacer si el contenido de un elemento HTML es más grande que el tamaño del elemento. Hay varios valores que se pueden utilizar con la propiedad overflow, cada uno con un comportamiento diferente. A continuación veremos algunos de los valores más comunes de la propiedad overflow.

visible

Permite que el contenido de un elemento HTML sea visible fuera del tamaño del elemento. Esto sirve para que el contenido se muestre fuera del tamaño del elemento. 

                        
                            .caja-uno {
                                overflow: visible;
                            }

hidden

Permite ocultar el contenido de un elemento HTML fuera del tamaño del elemento. Esto sirve para que el contenido no se muestre fuera del tamaño del elemento. 

                        
                            .caja-dos {
                                overflow: hidden;
                            }

scroll

Permite agregar barras de desplazamiento al contenido de un elemento HTML si el contenido es más grande que el tamaño del elemento. Esto sirve para que el contenido se muestre con barras de desplazamiento si es más grande que el tamaño del elemento. 

                        
                            .caja-tres {
                                overflow: scroll;
                            }

auto

Permite agregar barras de desplazamiento al contenido de un elemento HTML si el contenido es más grande que el tamaño del elemento. Esto sirve para que el contenido se muestre con barras de desplazamiento si es más grande que el tamaño del elemento. A diferencia de scroll en overflow, auto en overflow solo muestra las barras de desplazamiento si el contenido es más grande que el tamaño del elemento. 

                        
                            .caja-cuatro {
                                overflow: auto;
                            }

Otra manera de aplicar overflow es de manera individual a los lados de la caja. Por ejemplo si queremos aplicar overflow a la parte superior de la caja podemos hacerlo de la siguiente manera: 

                    
                        .caja-cinco {
                            overflow-x: hidden;
                        }
                    
                
                    
                        .caja-seis {
                            overflow-y: hidden;
                        }

Propiedad box-sizing

La propiedad box-sizing se utiliza para especificar cómo se calculará el tamaño de un elemento HTML. Hay dos valores que se pueden utilizar con la propiedad box-sizing: content-box y border-box. A continuación veremos cómo se utilizan estos valores en la propiedad box-sizing.

content-box

La propiedad box-sizing: content-box se utiliza para especificar que el tamaño de un elemento HTML se calculará incluyendo el contenido, pero no el relleno, el borde ni el margen. Esto sirve para que el tamaño del elemento se calcule incluyendo solo el contenido. Este es el valor por defecto que tienen los elementos HTML. 

                        
                            .caja-uno {
                                box-sizing: content-box;
                            }

border-box

La propiedad box-sizing: border-box se utiliza para especificar que el tamaño de un elemento HTML se calculará incluyendo el contenido, el relleno y el borde, pero no el margen. Esto sirve para que el tamaño del elemento se calcule incluyendo el contenido, el relleno y el borde. Desde hace tiempo se recomienda utilizar este valor para la propiedad box-sizing reseteando los estilos de los elementos HTML. 

                        
                            .caja-dos {
                                box-sizing: border-box;
                            }

Colapso de Margenes en Vertical

El colapso de márgenes en vertical es un fenómeno que ocurre cuando dos márgenes verticales se superponen. Esto puede causar problemas de alineación y espaciado en la página. A continuación veremos cómo se produce el colapso de márgenes en vertical y cómo se puede evitar.

¿Cómo se produce este colapso de margen vertical?

Se produce cuando dos márgenes verticales se superponen. Esto puede ocurrir cuando dos elementos se colocan uno encima del otro y tienen márgenes inferiores y superiores respectivamente. En este caso, los márgenes se superponen y se produce el colapso de márgenes en vertical. 

                            
                                .caja-uno {
                                    margin-bottom: 20px;
                                }
                                .caja-dos {
                                    margin-top: 20px;
                                }

En este caso, los márgenes de la caja-uno y la caja-dos se superponen y se produce el colapso de márgenes en vertical. El margen entre ellos sería de 20px en lugar de 40px.

¿Cómo podemos evitarlo?

La mejor opción es simplemente evitando usar los dos margenes simultamente.

En los casos de los margenes horizontal no se produce el colapso de margenes. Estos se suman entre si.
