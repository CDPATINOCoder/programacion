Tablas en CSS

Las tablas en HTML son una forma de mostrar información en forma de filas y columnas. En CSS podemos modificar el aspecto de las tablas utilizando las propiedades de la tabla.

Propiedades de tablas

Las propiedades de las tablas en CSS son las siguientes:

  • border-collapse: esta propiedad define si los bordes de las celdas de la tabla se colapsan o no. Los valores posibles son collapse y separate.
  • border-spacing: esta propiedad define el espacio entre las celdas de la tabla.
  • caption-side: esta propiedad define la posición de la leyenda de la tabla. Los valores posibles son top y bottom.
  • empty-cells: esta propiedad define si se muestran o no las celdas vacías de la tabla. Los valores posibles son show y hide.
  • table-layout: esta propiedad define cómo se distribuyen las celdas de la tabla. Los valores posibles son auto, fixed y inherit.
