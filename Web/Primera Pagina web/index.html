HTML

HTML, que significa "Lenguaje de marcado de Hipertexto" en inglés (Hypertext Markup Language), es el lenguaje estándar utilizado para crear y diseñar páginas web. Es un lenguaje de marcado que define la estructura y el contenido de una página web utilizando una serie de etiquetas y elementos.

El propósito principal de HTML es organizar el contenido de una página web de una manera estructurada y semántica, lo que significa que cada elemento HTML describe el tipo de contenido que contiene (por ejemplo. párrafos, encabezados, listas, enlaces, imágenes, formularios, etc.). Esto permite a los navegadores web interpretar correctamente la página y mostrarla de manera adecuada al usuario. HTML se compone de una serie de elementos que están delimitados por etiquetas. Cada etiqueta tiene una función específica y ppuede contener atributos para proporcionar información adicional sobre el elemento. Por ejemplo, la etiqueta "<p>" se utiliza para definir un párrafo, y la etiqueta "<img>" se utiliza para insertar imágenes en la página.

En resumen, HTML es el lenguaje fundamental que se utiliza para crear páginas web y proporciona la estructrua básica y el contenido de una página, lo que permite a los desarrolladores web crear experiencias interactivas y multimedia para los usuarios en la web.

Tipos de etiqueta HTML

  1. Etiquetas de texto básica:

    2. Son utlizadas para resaltar textos de manera visible, estos son algunos de ellos;

    1. <p> </p>: Sirve para escribir un párrafo de texto.
      <p>Esto es un párrafo</p>
    2. <b> </b>: Sirve para escribir un texto en negrita.
      <b>negrita</b>
    3. <i> </i>: Sive para escribir texto en itálica.
      <i>itálica</i>
    4. <u> </u>: Sirve para subrayar texto.
      <u>subrayado</u>
    5. <sup> <sup/b>: Sirve para expresar potencia:
      <sup>22</sup>
    6. <sub> </sub>: Sirve para colocar sub indice.
      H<sub>2</sub>O = H2O
    7. <mark> </mark>: Sirve para marcar Texto.
      <mark>Marcatexto</mark>
    8. <small> </small>: Sirve para que el texto escritó salga pequeño.
      <small>Este texto se verá pequeño</small> = Este texto se verá pequeño

  2. Etiquetas de texto semántica:

    3. Son utilizadas para resaltar palabras o textos que son importantes también de manera visual, estos son algunos de ellos;

    1. <strong> </strong>: Sirve para resaltar un texto indicando que es importante "negrita"
      <strong>texto importante</strong> = texto importante
    2. <em> </em>: Sirve para hacer enfásis en un texto.
      <strong>enfasis</strong>= enfasis
    3. <blockquote> </blockquote>: Sirve para definir una cita larga.
      <blockquote>La inteligencia de una persona no solo se mide a nivel cognitivo.</blockquote> =
      La inteligencia de una persona no solo se mide a nivel congnitivo.
    4. <cite> </cite>: Sirve para definir la fuente de cierta aclaración.
      <cite>César Patiño</cite> =
      La inteligencia de una persona no solo se mide a nivel cognitivo
      César Patiño

  3. Etiquetas de salto:

    4. Son utilizadas para hacer saltos, como dar un enter en word. Estos son algunos de ellos;

    1. <br>: Sirve para dar un salto de linea, puedes hacer más de un salto de linea, abajo vemos el uso.

      Lorem ipsum dolor sit amet consectetur adipisicing elit. <br>
      Porro eius vitae cupiditate architecto hic sapiente veniam, assumenda dolorum ut deleniti, asperiores obcaecati maiores aliquam laborum ab voluptate est modi vel!

  4. Etiquetas de Formateo:

    5. Se utilizan para presentar y formatear texto de una manera específica que refleje su significado o propósito dentro del contenido de la página web. Estos son algunos de ellos;

    1. <pre> </pre>: Esta etiqueta se utiliza para mostrar texto preformateado, conservando espacios en blanco y saltos de línea tal como están escritos en el código HTML. 
      <pre>
aquí no dejo espacios
    aquí hice una tabulación
            aquí hice dos tabulaciones
</pre>
    2. <code> </code>: Se utiliza para representar código fuente o fragmentos de código dentro de un párrafo de texto.
      <code>console.log("Hello, world!");</code> = console.log("Hello, world!");

  5. Etiquetas de semánticas estructurales:

    6. Son etiquetas para definir el tipo de maquetación que tendrá el sitio web, estas etiquetas pueden ir una dentro de otra. Estas son algunas de ellas:

    1. <header> </header>: Es el encabezado de una página web, generalmente va información acorde al tema general como el titulo de la pagina o tambien menús interactivos, infos de contacto, etc. 
                                  
<header>
    <h1>HTML</h1>
    <nav>
        <a>Inicio</a>   
        <a>Acerca De</a>   
        <a>Contacto</a>   
    </nav>
</header>
    2. <div> </div>: Es una etiqueta contenedora que no tiene semántica.
    3. <main> </main>: Define la sección principal del documento, aquí va el contenido que quieres mostrar en la web, sólo puede existir una etiqueta main por documento. 
                                  
<main>
    <h1>¿Qué es HTML?</h1>
    <p>
    HTML es un lenguaje de marcado que posibilita definir la estructura de nuestro documento mediante etiquetas
    </p>
    
</main>

    5. <nav> </nav>: Sirve para representar una navegación.

    6. <article> </article>: Sirve para representar una sección de autocontenido (se explica por si sola).

  6. Etiquetas de bloque VS Etiquetas de linea

    7. La etiqueta de bloque por preferencia es la <div>, una etiqueta de bloque es aquella que genera un salto de linea automaticamente luego de cerrarse.

    Y la etiqueta de linea por excelencia es la <span>, una etiqueta de linea sólo ocupa el espacio necesario que tiene su contenido.

  7. Imágenes

    8. Esta etiqueta no contiene una de cierre, sirve para adjuntar una imagen dentro de la página web, lleva atributos como: "src" y "alt". La manera de usar es <img src="ruta de la imagen" alt="nombre de la imagen">

  8. Figuras

    9. Es una etiqueta de tipo semántica, sirve para adjuntar generalmente imagenes que hacen referencia al contenido del que se está hablando. Esta es la manera de uso: 

                        
<figure>
    <img src="ruta de la imagen" alt="nombre descriptivo">
    <figcaption>nombre descriptivo</figcaption>
</figure>

  9. Listas

    Existen dos tipos de listas en HTML:

    1. Listas Ordenadas: Se expresan de esta manera <ol> </ol>, esta etiqueta ordena de manera predeterminada del 1 hasta N el cual es la cantidad de elementos que vas a citar, pueden tener atributos como type="A" (ordenar De A hasta indefinido), reversed(se ordena de mayor a menor), etc.

      Como ejemplo podemos tener: 

                                          
<ol>
    <li>Tema 1</li>
    <li>Tema 2</li>
    <li>Tema 3</li>
    <li>Tema 4</li>
</ol>

    2. Listas Desordenadas: Se expresan de esta manera <ul> </ul>, esta etiqueta ordena de manera predeterminada con una viñeta, puede tener atributos como type="disc" (viñeta), type="square" (viñeta cuadrada), type="circle" (viñeta circulo sin relleno en negro), etc.

      Como ejemplo podemos tener: 

                                          
<ul>
    <li>Tema </li>
    <li>Tema </li>
    <li>Tema </li>
    <li>Tema </li>
</ul>

  10. videos

    Los videos en una página web funcionan casi como la etiqueta img, se coloca de esta manera <video src="ruta del video">

    podemos utilizar atributos como autoplay (para que se reproduzca de manera automatica), muted (para que no se reproduzca con audio), controls (para acceder a los controles del video como play, volumen o pantalla completa), loop (para que al terminar el video se reproduzca de nuevo), etc.

  11. audios

    Los videos en html también funcionan similar a un video o una img, de esta manera podemos invocarlo <audio src="ruta del audio">

    Los audios por defecto no se ven dentro de la página web, solo se escucha, si queremos ver el audio o acceder a los controles lo hacemos también con el atributo control. Probamos con pistas de audios.
